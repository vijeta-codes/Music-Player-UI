# Music-Player-UI
A modern responsive music player UI built using HTML, CSS, Flexbox, and Grid.

## Features
- Fixed sidebar navigation
- Responsive hero section
- Trending and recent songs grid
- Fixed bottom player bar
- Hover effects and smooth transitions
- Responsive design using media queries

## Built With
- HTML5
- CSS3
- Flexbox
- CSS Grid

## What I Learned
- Layout architecture
- Flexbox and Grid
- Responsive design
- Image handling using object-fit
- Fixed positioning
